Quick Start — Create a New Dashboard

Three steps to spin up a new dashboard from the blank template:

# 1. Copy the blank template cp templates/blank.html dashboards/my-new-dashboard/index.html # 2. Update the paths in the new file (two lines to change) # href="../../shared/css/design-system.css" # src="../../shared/js/dashboard-utils.js" # 3. Open in browser — no build step required open dashboards/my-new-dashboard/index.html
